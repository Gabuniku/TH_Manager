■初めに
　このソフトは上海アリス幻樂団より制作されている東方Projectの作品の実行ファイルを
　まとめて管理し、1つの画面で実行設定等ができるソフトとなっています。
　皆さんの東方シューターライフのお役に立てれば幸いです。

■インストール
　基本的にお好きなところに配置していただいて構いません。

■アンインストール
　実行ファイルを消してもらえば完了です。

■簡易説明
　・メニューバー
　　☆ファイル
　　　〇追加
　　　　東方の実行ファイル（th〇〇.exe）を選択してください。

　　　〇終了
　　　　終了します。

　・上部ボタン
　　☆追加
　　　メニューバー参照

　　☆フォルダー内を検索
　　　選択したフォルダーにある東方の実行ファイルを探して登録します。（子フォルダーも参照します）

　　☆昇順に並び替え
　　　登録された東方作品を作品ナンバリングをもとに昇順に並び替えます。

　・リスト
　　☆起動
　　　起動します。

　　☆設定
　　　custom.exeを起動します。

　　☆フォルダーを開く
　　　実行ファイルがあるフォルダーを開きます。

　　☆セーブフォルダーを開く
　　　セーブデータがあるフォルダーを開きます。

　　☆一覧から削除
　　　一覧から削除します。
